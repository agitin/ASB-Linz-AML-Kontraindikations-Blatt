# Schwere anaphylaktische Reaktion

## Symptome

Ereignisbezogener Zusammenhang (Insektenstich, Medikamente, Nahrungsmittel, etc.) Schwere Atemnot, Stridor, Bronchospasmus oder Schocksymptome eventuell Hautsymptome

## AML1

### Kinderanwendung (AML1)

**Notarzt:** alarmieren

**1. Epinephrin**, Handelsname: EpiPen Junior

Kontraindikationen: Allergie / Unverträglichkeit, unter 15 kg KG / 3. LJ

| Gewichts-/Altersklasse | Dosierung | Applikation |
| -------- | ------- | ------- |
| 15–30 kg KG / 3.–12. LJ | 150 µg | i.m. Autoinjektor |
| über 30 kg KG / über 12. LJ | 300 µg | i.m. Autoinjektor |

### Erwachsenenanwendung (AML1)

**Notarzt:** alarmieren

**1. Epinephrin**, Handelsname: EpiPen

Kontraindikationen: Allergie / Unverträglichkeit

| Gewichts-/Altersklasse | Dosierung | Applikation |
| -------- | ------- | ------- |
| keine | 300 µg | i.m. Autoinjektor |

## AML2

**Notarzt:** alarmieren

**1. Epinephrin**, Handelsname: Adrenalin Pädia

Kontraindikationen: Allergie / Unverträglichkeit

| Gewichts-/Altersklasse | Dosierung | Applikation |
| -------- | ------- | ------- |
| unter 50 kg KG | 10 µg / kg KG (=0,05 ml / kg KG) | i.m. |
| über 50 kg KG | 500 µg (0,5 ml) | i.m. |
