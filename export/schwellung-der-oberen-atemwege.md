# Schwellung der oberen Atemwege

## Symptome

**Ganzer Text:** Ereignisbezogener Zusammenhang (Inhalationstrauma, Insektenstich im Mundbereich, allergische Atemwegsschwellung oder allergischer Bronchospasmus) Schwellung der oberen Atemwege, Stridor, Giemen, Brummen, Pfeifen, Heiserkeit, Atemnot, Atemhilfsmuskulatur, erhöhte Atemfrequenz, Krupp-Syndrom

 **Hervorgehoben:** Ereignisbezogener Zusammenhang, Schwellung der oberen Atemwege, Stridor, Atemnot

 **Gekürzt:** Das ist eine gekürzte Version der Symptome

## AML1

### Kinderanwendung (AML1)

**Notarzt:** alarmieren *(Kommentar: ASB Linz)*

#### Wirkstoff: Epinephrin

Handelsname: Adrenalin Pädia

Kontraindikationen: Allergie / Unverträglichkeit, unter 15 kg KG / 8. LJ, Tachy/arrhythmie > 120/min, kardio-pulmonale Risikofaktoren *(Kommentar: ASB Linz)*

| Gewichts-/Altersklasse | Dosierung | Applikation | Kommentar |
| -------- | ------- | ------- | ------- |
| keine | 3 mg (=3 ml) | inhalativ via O2-Verneblermaske |  |

Wiederholung: keine *(Kommentar: ASB Linz)*
