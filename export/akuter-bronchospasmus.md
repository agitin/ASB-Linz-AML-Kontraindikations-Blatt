# Akuter Bronchospasmus

## Symptome

Akute Atemnot, erhöhte Atemfrequenz, Atemgeräusche: Giemen, Brummen, Pfeifen, verlängertes Exspirium

## AML1

### Erwachsenenanwendung (AML1)

**Notarzt:** verständigen

**1. Salbutamol/Ipratropium**, Handelsname: Combivent

Kontraindikationen: Allergie / Unverträglichkeit, < 18. LJ, HF > 140/min, Salb./Ipratr. bereits in den letzten 4h eingenommen

| Gewichts-/Altersklasse | Dosierung | Applikation |
| -------- | ------- | ------- |
| keine | 2,5 ml | inhalativ via O2-Verneblermaske |

## AML2

### Erwachsenenanwendung (AML2)

**Notarzt:** verständigen

Kontraindikationen: Allergie / Unverträglichkeit, < 18. LJ, HF > 140/min, Salb./Ipratr. bereits in den letzten 4h eingenommen

| Gewichts-/Altersklasse | Dosierung | Applikation |

Kontraindikationen: Allergie / Unverträglichkeit

| Gewichts-/Altersklasse | Dosierung | Applikation |
