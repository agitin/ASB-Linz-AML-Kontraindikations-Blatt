# Akuter Bronchospasmus

## Symptome

**Ganzer Text:** Akute Atemnot, erhöhte Atemfrequenz, Atemgeräusche: Giemen, Brummen, Pfeifen, verlängertes Exspirium

 **Hervorgehoben:** Atemnot, erhöhte Atemfrequenz, Giemen, Brummen, Pfeifen, verlängertes Exspirium

 **Gekürzt:** Das ist eine gekürzte Version der Symptome

## AML1

### Erwachsenenanwendung (AML1)

**Notarzt:** verständigen

#### Wirkstoff: Salbutamol/Ipratropium

Handelsname: Combivent

Kontraindikationen: Allergie / Unverträglichkeit, < 18. LJ, HF > 140/min, Salb./Ipratr. bereits in den letzten 4h eingenommen

| Gewichts-/Altersklasse | Dosierung | Applikation | Kommentar |
| -------- | ------- | ------- | ------- |
| keine | 2,5 ml | inhalativ via O2-Verneblermaske |  |

Wiederholung: keine
